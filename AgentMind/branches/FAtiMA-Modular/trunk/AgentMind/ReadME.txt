This folder contains:


 - .metadata information that allows you to open the 
    source code as an Eclipse 3.0 Project
    
    To open it in eclipse, checkout the whole directory (FearNotAgent-V1),
    start Eclipse 3.0 and switch the workspace to be the directory you 
    checked out. 

 - the source code for FAtiMA (FearNot AffecTIve Mind Architecture)

   If you want to work only with the source code, or not use Eclipse 3.0,
   checkout the directory FAtiMA/src
